# Feature: Plugin Rampes et Garde-corps

## Description
Calculateur de rampes et garde-corps conforme au Code de construction du Québec.

## Entrées
- Longueur de la rampe (mm)
- Hauteur de chute (mm)
- Type d'usage : résidentiel privé / résidentiel commun / commercial
- Matériau : bois / métal / verre / câble
- Type d'installation : escalier / balcon / terrasse

## Sorties
- Hauteur de garde-corps requise (mm)
- Espacement barreaux conforme (≤ 100 mm)
- Nombre de poteaux
- Nombre de barreaux
- Longueur de main courante (mm, avec dépassements)
- Quantité de matériaux (poteaux, barreaux, main courante)
- Plan de pose étape par étape
- Visualisation 3D
- Indicateurs de conformité CCQ

## Normes applicables
- CCQ 9.8.7.4 : hauteur main courante 800–965 mm
- CCQ 9.8.8.1 : hauteur garde-corps selon hauteur de chute
- CCQ 9.8.8.3 : espacement barreaux ≤ 100 mm
