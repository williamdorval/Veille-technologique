# Feature: Plugin Calcul de plancher

## Description
Outil de dimensionnement de solives de plancher selon la portée, l'usage et le type de bois.

## Entrées
- Portée des solives (mm)
- Largeur de la pièce (mm)
- Type d'usage : chambre / salon / salle de bain / garage / commercial
- Type de bois : SPF / sapin de Douglas / LVL
- Type de sous-plancher : OSB / contreplaqué
- Présence d'élément lourd (baignoire, piano…)

## Sorties
- Dimension de solive recommandée (2x6 à 2x12)
- Espacement optimal (300/400/600 mm)
- Nombre de solives et longueur totale de bois
- Quantité de panneaux de sous-plancher
- Flèche calculée vs flèche maximale admissible (L/360)
- Indicateurs de conformité CNB/CCQ
- Visualisation 3D structure de solives

## Normes applicables
- CNB 2020, Table 4.1.5.3 : charges vives résidentielles (1,9 kPa)
- CCQ 9.4.3.1 : flèche admissible L/360
- CCQ 9.23.15.3 : sous-plancher
