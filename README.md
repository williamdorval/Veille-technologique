# Plateforme web d'aide aux constructeurs

Site web modulaire qui regroupe des outils pratiques pour les constructeurs québécois. Premier plugin : calculateur d'escaliers conforme aux normes du Code de construction du Québec.

Projet réalisé dans le cadre du cours **Veille technologique (420-1SH-SW)** du Cégep de Shawinigan, volet 1 — Specification Driven Development.

## Aperçu

| Élément | Valeur |
|---|---|
| Étudiant | William Dorval |
| Méthode | Specification Driven Development (SDD) avec SpecKit |
| Agent IA | Claude Code |
| Stack | Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui |
| Wiki | Obsidian (dossier `docs/`) |

## Pour lancer en local

```bash
npm install
npm run dev
```

Puis ouvrir [http://localhost:3000](http://localhost:3000).

## Documentation (wiki)

Ouvrir le dossier `docs/` comme vault dans Obsidian, puis commencer par `docs/index.md`.

## Avertissement

Cette plateforme est un outil d'aide à la conception et **ne remplace pas** la consultation d'un professionnel certifié.
