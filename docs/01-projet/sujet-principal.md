# Sujet principal

## Le projet

Une plateforme web modulaire d'aide aux constructeurs québécois. La plateforme accueille des outils sous forme de plugins. Le MVP livre :

- Une page d'accueil de la plateforme
- Une structure modulaire qui permet d'ajouter d'autres plugins plus tard sans tout casser
- Un premier plugin fonctionnel : calculateur d'escaliers selon les normes du Québec

## Le thème du cours

« L'IA au service du développeur ». Le projet illustre ce thème par sa méthode : préparer un contexte rigoureux (spécification, plan, tâches, documentation métier) puis laisser Claude Code implémenter l'application.

## Le projet réalisable en 7 jours

Le périmètre est volontairement borné. Un seul plugin (escaliers) dans le MVP. Pas de backend, pas de base de données, pas d'authentification. Tout fonctionne en local.

## L'approche Specification Driven Development

Le projet utilise SpecKit pour structurer le workflow SDD en quatre phases : Spec → Plan → Tasks → Implement.
