# Limites du projet

## Un seul plugin dans le MVP

Le MVP se limite au calculateur d'escaliers.

## Phase de préparation plus longue qu'en vibe coding

Le SDD demande une phase de spécification et de documentation avant de pouvoir écrire du code.

## Qualité dépend de la documentation préparée

Si la documentation des normes est incomplète, Claude Code va faire des erreurs.

## Pas un remplacement à un professionnel certifié

Le calculateur respecte les normes documentées, mais l'utilisateur reste responsable de la conformité finale sur le chantier.

## Pas d'IA intégrée dans le calculateur (pour l'instant)

Algorithme mathématique déterministe pour le MVP. Intégration d'une IA spécialisée envisagée pour une version future.
