# Avantages de l'approche

## Code IA prévisible et conforme aux specs

Avec une spec claire, Claude Code arrête d'inventer des comportements. Il fait ce qui est écrit.

## Méthode reproductible pour d'autres projets

Une fois la méthode maîtrisée, elle est applicable à tous les futurs projets.

## Documentation réutilisable et structurée

Le wiki Obsidian produit n'est pas jetable. Les pages sur les normes du Québec peuvent servir à d'autres plugins (rampes, planchers, structures) dans des versions futures.

## Livrable concret et utile

Contrairement à beaucoup de projets scolaires, celui-ci produit un outil avec un vrai cas d'usage.
