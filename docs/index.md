# Plateforme d'outils pour constructeurs québécois

**Cours** : 420-1SH-SW — Développement Web | **Étudiant** : William Dorval | **Cégep Shawinigan**

---

Cette plateforme regroupe des calculateurs spécialisés pour les professionnels de la construction résidentielle au Québec. Chaque outil vérifie la conformité avec le Code de construction du Québec (CCQ) et le Code national du bâtiment (CNB 2020).

## Plugins disponibles

| Plugin | Description |
|--------|-------------|
| **Escaliers** | Calcul conforme CCQ — contremarche, giron, Blondel, dégagement, garde-corps |
| **Rampes et garde-corps** | 4 matériaux, normes privé/commun, plan de construction |
| **Plancher** | Dimensionnement de solives — Euler-Bernoulli, SPF/Douglas/LVL |
| **Toiture** | Surfaces, chevrons, ventilation, charges de neige par région QC |
| **Analyse de plan (IA)** | Lecture de croquis manuscrits par Gemini → export Excel |
