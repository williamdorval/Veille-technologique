# Skills Claude Code

Ce dossier contient les skills installés pour ce projet.

## Comment installer les skills SpecKit

Au début du projet, lancer dans le terminal :

```bash
uvx --from git+https://github.com/github/spec-kit.git specify init --here --ai claude --integration-options="--skills"
```

Cela ajoute automatiquement les skills SpecKit dans ce dossier.
